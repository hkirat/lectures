Basic Terminology

Tag

Used to tag or "mark-up" pieces of text. Once tagged, the text becomes HTML code to be interpreted by a web browser. Tags look like this: <tag>

example
Opening tag: <body>
Closing tag: </body>

Element

A complete tag, having an opening <tag> and a closing </tag>

example: <p>This is a paragraph</p>

Attribute

Used to modify the value of the HTML element. Elements will often have multiple attributes.

Comments

  • Comment a lot
  • Comment sensibly
  • Make your life easy


syntax: <!-- enter your amazing comment here -->

Elements

When you land on a website, all the items you see in front of you -- the paragraph texts, the page banners, and the navigation links are all elements of the web page.

Four critical elements:

  1. HTML
  2. head
  3. title
  4. body

HTML

<html> is the element that begins and ends each and every web page.

HEAD

Contains elements that are not displayed by the browser.

Notice the title tag inside head.

BODY

Encapsulates all the visual elements (paragraphs, pictures, tables, etc.) of a web page.

Tags

  • Paragraph: <p> .... </p>
  • heading: <h1> .... </h1>
  • Unordered list: <ul> .... </ul>
  • Image: <img />
  • Break: <br />

Anchor tags

<a>anything here</a>

<a href="http://google.com">My link to google</a>

gets converted to My link to google

Absolute links

eg: http://google.com


Relative links

eg: ../img/penguin.png
./img/sds.png
