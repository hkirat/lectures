Recursive Functions

A function is said to be recursive if it calls itself.

Fibonacci series

f(0) = 0
f(1) = 1

f(n) = f(n-1) + f(n-2) for n > 1

0, 1, 1, 2, 3, 5, 8, 13...

Method

This is the pseudocode for calculating Nth fibonacci number. 

function fibo(int n)
  if n < 2:
    return n
  else:
    return fibo(n-1) + fibo(n-2)
end fibo

Power Function

Calculating the value of an

Pseudocode

function power(int a, int n)
  if n == 0:
    return 1
  else:
    return power(a, n - 1) * a
end power

Process

In both the examples, we check for a particular case and end the process. Or else we repeat the process of calculation.

Better way of Calculating an

Pseudocode 1

function power(int a, int n)
  if n == 0:
    return 1
  else:
    if n is even:
      return power(a, n/2) * power(a, n/2)
    else:
      return a * power(a, n/2) * power(a, n/2)
end power

Pseudocode 2

function power(int a, int n)
  if n == 0:
    return 1
  else:
    int x = power(a, n/2)
    if n is even:
      return x * x
    else:
      return a * x * x
end power

Dynamic Programming

What is Dynamic Programming?

It is a technique used to solve a problem by first solving smaller sub-problems.

Two Approaches

  • Bottom-Up
  • Top-down

Example

How will you calculate the value of N! (N factorial)?


How will you calculate for many such values of N?

Say N = 5, 9, 24 and 30.

Naive Approach

For each value of N, you will find the answer by multiplying all the numbers from 1 to N.

Using Dynamic Programming

Rather than calculating the value of N! for each N, we store the result as we calculate the value for the maximum value of N (30 in this case).

Thus we have stored the value of 1!, 2!, 3!, ..., N!.

Bottom-Up

fibo[0] = 0
fibo[1] = 1

for i = 2, 3, 4, ... n:
  fibo[i] = fibo[i-1] + fibo[i-2]

Top-Down

function fibo(int n)
  if n < 2:
    return n
  else if mem_fibo[n] is already stored:
    return mem_fibo[n]
   else:
    mem_fibo[n] = fibo(n-1) + fibo(n-2)
