Document Object Model (DOM)

The DOM defines a standard way for accessing and manipulating HTML documents.

Using layout engines, a web browser converts HTML to DOM.

This DOM provides methods to manipulate the data.

DOM Nodes

Everything in an html document is represented as a node. These nodes share a basic hierarchical relationship.

Manipulating the DOM

Javascript provides access to DOM and allows to manipulate it. This includes stuff like:

document.getElementById('special').onclick = someCoolFunction();

$ - jQuery function

The Flow:

  • You pass a css selector to the "$" function
  • This returns a DOM object which has various methods and events
  • We add event listeners for various possible events for the DOM
  • These event listeners call functions (known as callbacks) whenever these events are triggered

var sum = function (a, b, callback) {
  callback(a + b);
};
sum(10, 20, function (s) {
  $('#sum').append(s);

A Practical use for Form Validation

Say you have a simple form

<form id="registration">
  Name: <input id="name" type="text"><br>
  Email: <input id="email" type="text"><br>
  Password: <input id="pass" type="password"><br>
  Confirm Password: <input id="cpass" type="password"><br>
  <input type="submit" value="Submit">
</form>

The Code 

$('#registration').on('submit', function () {
  // Get all the details from the form
  var name = $('#name').val();
  var email = $('#email').val();
  var pass = $('#pass').val();
  var cpass = $('#cpass').val();

  // Check for full name
  if (name.indexOf(' ') == -1) {
    alert('Please enter full name!');
    return false;
  }
  // Check whether both passwords match
  else if (pass != cpass) {
    alert('Both passwords do not match!');
    return false;
  }
});
