The Flow

The main process for content manipulation

  • Use css selectors to specify an html element
  • This javascript object has various properties and functions associated with it
  • Add various event listeners
  • Specify a function that is called whenever an event is triggered

Document Object Model (DOM)

Treat everything in the HTML document as an object

  • All web browsers use a layout engine to convert the html document into html elements
  • It adds various properties and functions for each element to manipulate the style, the data and to add events
  • The browser then allows javascript to access these elements and use them as objects

A Practical use for Form Validation

Say you have a simple form

<form id="registration">
  Name: <input id="name" type="text"><br>
  Email: <input id="email" type="text"><br>
  Password: <input id="pass" type="password"><br>
  Confirm Password: <input id="cpass" type="password"><br>
  <input type="submit" value="Submit">
</form>

The Code 

$('#registration').on('submit', function (event) {
  // Get all the details from the form
  var name = $('#name').val();
  var email = $('#email').val();
  var pass = $('#pass').val();
  var cpass = $('#cpass').val();

  // Check for full name
  if (name.indexOf(' ') == -1) {
    alert('Please enter full name!');
    event.preventDefault();
  }
  // Check whether both passwords match
  else if (pass != cpass) {
    alert('Both passwords do not match!');
    event.preventDefault();
  }
});
